# دليل البدء السريع - نظام الطباعة المحسّن

## للمستخدمين

### الوصول إلى إعدادات الطباعة

1. افتح التطبيق
2. اذهب إلى **الإعدادات** → **إعدادات الطباعة**
3. ستظهر لك واجهة بـ 6 تبويبات

### ضبط الإعدادات

#### معلومات المؤسسة
- أدخل اسم الجامعة والكلية والقسم
- هذه المعلومات ستظهر في رأس كل مستند مطبوع

#### الشعارات
- اضغط على "تحميل شعار الجامعة" أو "تحميل شعار الكلية"
- اختر صورة من جهازك (PNG, JPG, SVG)
- اضبط حجم الشعار باستخدام المنزلق (40-200 بكسل)

#### أحجام الخطوط
- اضبط حجم خط الترويسة (8-24 نقطة)
- اضبط حجم خط العنوان (10-32 نقطة)
- اضبط حجم خط محتوى الجداول (6-18 نقطة)

#### تنسيق الجداول
- اضبط المسافة الداخلية للخلايا (0-20 بكسل)
- اضبط ارتفاع السطر (1-3)
- اختر محاذاة النص (يمين/وسط/يسار)
- اضبط عرض ولون حدود الجدول

#### هوامش الصفحة
- اضبط الهوامش الأربعة (0-50 ملم)
- اختر حجم الصفحة (A4, A3, Letter, Legal)

#### إعدادات إضافية
- فعّل/عطّل إظهار أرقام الصفحات
- فعّل/عطّل إظهار تاريخ الطباعة
- أضف علامة مائية (اختياري)

### المعاينة المباشرة

- اضغط على زر "معاينة" لرؤية التغييرات فوراً
- ستظهر نافذة معاينة بجانب الإعدادات
- التغييرات تظهر مباشرة دون حاجة للحفظ

### حفظ الإعدادات

1. بعد ضبط جميع الإعدادات
2. اضغط على زر "حفظ"
3. ستظهر رسالة تأكيد
4. الإعدادات ستُطبق على جميع عمليات الطباعة

### إعادة التعيين

- اضغط على زر "إعادة تعيين" لاستعادة القيم الافتراضية
- سيطلب منك التأكيد قبل الإعادة

## للمطورين

### استخدام Hook الموحد

```typescript
import { usePrintSettings } from '../hooks/usePrintSettings';

function MyComponent() {
  const { settings, isLoading, saveSettings } = usePrintSettings();
  
  // استخدم settings في الطباعة
  console.log(settings.cellPadding); // 3
}
```

### طباعة مستند

```typescript
import { printContent } from '../utils/printUtils';
import { createPrintOptions } from '../hooks/usePrintSettings';

async function printMyDocument() {
  const settings = await getPrintSettingsWithDefaults();
  const htmlContent = '<h1>مرحباً</h1><table>...</table>';
  
  const options = createPrintOptions(
    'عنوان المستند',
    settings,
    { 
      orientation: 'landscape',
      asPDF: true 
    }
  );
  
  printContent(htmlContent, options);
}
```

### توليد PDF

```typescript
// الطريقة 1: استخدام asPDF
printContent(htmlContent, { ...options, asPDF: true });

// الطريقة 2: استخدام generatePDFFromHTML مباشرة
import { generatePDFFromHTML } from '../utils/printUtils';
generatePDFFromHTML(htmlContent, options);
```

### تخصيص الإعدادات لطباعة معينة

```typescript
const customOptions = {
  ...settings,
  title: 'تقرير خاص',
  orientation: 'portrait',
  pageSize: 'A3',
  tableCellAlignment: 'right',
  watermark: 'سري',
  watermarkOpacity: 0.2
};

printContent(htmlContent, customOptions);
```

## نصائح وإرشادات

### للحصول على أفضل النتائج

1. **الشعارات**: استخدم صور عالية الدقة (300 DPI على الأقل)
2. **الخطوط**: اختر أحجام خطوط مناسبة للقراءة (10-12pt للمحتوى)
3. **الهوامش**: اترك هوامش كافية (5-10mm على الأقل)
4. **المحاذاة**: استخدم "وسط" للجداول الرسمية

### حل المشاكل الشائعة

**المشكلة**: الطباعة غير موسطة
- **الحل**: تحقق من إعداد "محاذاة محتوى الخلايا" في تبويب "تنسيق الجداول"

**المشكلة**: الشعار كبير جداً
- **الحل**: اضبط "حجم الشعار" في تبويب "الشعارات"

**المشكلة**: النص صغير جداً
- **الحل**: زد "حجم خط محتوى الخلايا" في تبويب "أحجام الخطوط"

**المشكلة**: الجدول يتقطع بين الصفحات
- **الحل**: قلل "المسافة الداخلية للخلايا" أو قلل "ارتفاع السطر"

## الأسئلة الشائعة

**س: هل يمكنني حفظ أكثر من مجموعة إعدادات؟**
ج: حالياً يتم حفظ مجموعة واحدة فقط. ميزة القوالب المتعددة قيد التطوير.

**س: هل تؤثر الإعدادات على جميع الطباعات؟**
ج: نعم، الإعدادات المحفوظة تُطبق على جميع عمليات الطباعة في التطبيق.

**س: كيف أعيد الإعدادات الافتراضية؟**
ج: اضغط على زر "إعادة تعيين" في أعلى الصفحة.

**س: هل يمكنني طباعة بدون شعارات؟**
ج: نعم، اترك حقول الشعارات فارغة.

**س: ما هي العلامة المائية؟**
ج: نص شفاف يظهر في خلفية المستند (مثل "مسودة" أو "سري").
